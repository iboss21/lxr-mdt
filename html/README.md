This folder contains the web-based user interface for LXR-MDT.

## Structure

- **index.html** - Main HTML structure
- **css/** - Stylesheets (1899 immersive design)
  - style.css - Main stylesheet with western/leather theme
- **js/** - JavaScript application logic
  - app.js - NUI communication and UI interactions
- **img/** - Images and graphics
- **sounds/** - Sound effects

## Design

LXR-MDT uses an **1899-era immersive design**:
- Leather and paper textures
- Gold/bronze color scheme
- Period-appropriate fonts (Crimson Text, IM Fell English)
- Western aesthetic
- Ink and handwriting styles

## Technology

- Pure HTML5/CSS3/JavaScript
- No frameworks (lightweight)
- NUI callbacks to Lua
- Responsive design
