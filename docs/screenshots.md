## ğŸ“¸ Taking Screenshots

### In-Game Screenshots
1. Press F12 (Steam) or Print Screen
2. Screenshots saved to:
   - Steam: `Steam/userdata/[ID]/760/remote/[AppID]/screenshots/`
   - Manual: Windows Screenshots folder

### Console Screenshots
1. Use Snipping Tool or similar
2. Capture entire console window
3. Ensure text is readable

### Code Screenshots
1. Use VS Code or editor
2. Set color scheme (Dark+ recommended)
3. Zoom for readability
4. Show branded comments
